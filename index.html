Project overview

IT support teams receive large volumes of tickets that must be reviewed and routed to the correct team. This project shows how machine learning can classify ticket text automatically to support faster triage and more consistent handling.

Problem

Manual ticket review takes time, creates inconsistency, and can delay assignment to the correct support queue.

Approach

Use natural language processing to transform ticket text into features, then train a classifier on labeled historical tickets.

Outcome

Predict the likely ticket topic early, enabling better routing, prioritization, and operational visibility.

How it works

The model takes ticket text, cleans and vectorizes it, learns topic patterns from labeled examples, and returns a predicted class for new incoming tickets.

Training workflow

  • Download the dataset in Colab
  • Load the ticket text and labels
  • Split data into training and test sets
  • Tokenize text and pass through BERT encoder
  • Fine-tune classification head on ticket topics
  • Evaluate with accuracy and classification metrics

Inference workflow

  • User submits a new ticket
  • Text is preprocessed the same way as training
  • Model predicts a topic group
  • Output can be shown on screen or sent to an API
  • Support teams can use it for triage or review

Model results

The fine-tuned BERT model achieved 85.27% accuracy on 9,568 test samples, correctly classifying IT service tickets into 8 distinct support categories using contextual language understanding.

Performance metrics

  • Accuracy: 85.27%
  • Correct predictions: 8,159 / 9,568
  • Model type: BERT with fine-tuned classification layer
  • Encoder: BERT (Bidirectional Encoder Representations from Transformers)
  • Training approach: Supervised transfer learning with cross-entropy loss

Dataset and model

The current training workflow uses a labeled IT service ticket dataset, where ticket text acts as the input feature and the topic group acts as the target label.

Input field

Document contains the ticket text used for model training.

Target field

Topic_group is the class label the model learns to predict.

Model choice

BERT is a pre-trained transformer model that captures bidirectional context in text, providing strong baseline performance for multi-class text classification and easily interpretable transfer learning results.
